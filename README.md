# Versionskontrolle Easy

Willkommen zu Versionskontrolle Easy!

## Übersicht
Versionskontrolle Easy ist ein benutzerfreundliches Tool, das Entwicklern hilft, ihre Projekte zu verwalten und Änderungen effizient zu verfolgen. Es ist ideal für kleine Teams und Einsteiger in die Softwareentwicklung.

## Funktionen
- **Einfache Benutzeroberfläche**: Intuitive Bedienung für schnelle Einarbeitung.
- **Änderungsverlauf**: Speichern und anzeigen von Änderungen im Zeitverlauf.
- **Zusammenarbeitstools**: Gemeinsames Arbeiten an Projekten mit einfachen Merge-Funktionen.
- **Branching**: Erstellen und Verwalten von Entwicklungszweigen leicht gemacht.

## Installation
```bash
# Klonen Sie das Repository
git clone https://github.com/benutzername/versionskontrolle-easy.git

# In das Verzeichnis wechseln
cd versionskontrolle-easy

# Abhängigkeiten installieren
npm install
```

## Nutzung
1. Ein neues Projekt initialisieren:
   ```bash
   vce init mein-projekt
   ```
2. Änderungen verfolgen:
   ```bash
   vce track meine-datei.txt
   ```
3. Änderungen hochladen:
   ```bash
   vce commit -m "Meine erste Änderung"
   ```
